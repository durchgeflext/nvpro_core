@DOC_START

struct nvgl::Buffer

Wraps buffer as well as optionally creates a GL_TEXTURE_BUFFER if a non-null format is provided. If bindless is available it will also create bindless handles for all resources and make them resident.

If the flags contain GL_MAP_PERSISTENT_BIT it will also map the buffer and keep the host pointer. @DOC_END

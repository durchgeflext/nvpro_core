class nvgl::AppWindowProfilerGL

nvgl::AppWindowProfilerGL derives from nvh::AppWindowProfiler and overrides the context and swapbuffer functions.

To influence the context creation modify m_contextInfo prior running AppWindowProfiler::run, which triggers window, and context creation etc.

The class comes with a nvgl::ProfilerGL instance that references the AppWindowProfiler::m_profiler's data. @DOC_END

nvgl resource functions

Functions that wrap glCreate/glDelete and operate on GLuint& obj. The "new" functions delete the existing object if non-null and create a new one. The "delete" functions delete non-null objects.

newBuffer / deleteBuffer newTextureView newTexture / deleteTexture newFramebuffer / deleteFramebuffer newSampler / deleteSampler newQuery / deleteQuery newVertexArray / deleteVertexArray

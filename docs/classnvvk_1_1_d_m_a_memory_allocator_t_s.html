class nvvk::DMAMemoryAllocatorTS

nvvk::DMAMemoryAllocatorTS is using nvvk::DeviceMemoryAllocator internally. It implements a simple thread-safe wrapper, not optimized for performance. nvvk::DeviceMemoryAllocator derives from nvvk::MemAllocator as well, so this class here is for those prefering a reduced wrapper; @DOC_END

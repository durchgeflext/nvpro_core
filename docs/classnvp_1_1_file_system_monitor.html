Detailed Description

@DOC_START

class nvp::FileSystemMonitor

Monitors files and/or directories for changes

This cross-platform wrapper does not create any threads, but is designed for it. checkEvents() will block until either an event is generated, or cancel() is called. See ModifiedFilesMonitor for an example. @DOC_END
